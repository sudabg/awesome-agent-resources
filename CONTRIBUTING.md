# Contributing to Awesome Agent Resources

## Adding a Resource

1. Fork this repo
2. Add your resource to the appropriate section in `README.md`
3. Keep the format: `- [Name](URL) - Short description. \`tags\``
4. Submit a pull request

## Format

```markdown
- [Resource Name](https://github.com/user/repo) - One-line description of what it does and why it matters. `Python` `MIT` ⭐1.2K
```

## Criteria

- ✅ Open source preferred (include license tag)
- ✅ Active development (updated in last 6 months)
- ✅ Clear documentation
- ✅ Relevant to AI agents, LLMs, or self-improvement
- ❌ No paywalled-only tools
- ❌ No pure marketing content

## Sections

Choose the most relevant section. If none fits, propose a new one.

## Tags

Use these tags where applicable:
- Language: `Python` `TypeScript` `Shell` `Rust`
- License: `MIT` `Apache-2.0` `CC-BY-4.0`
- Stars: `⭐1.2K` (optional, update periodically)
